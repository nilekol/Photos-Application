Photo Documentation

The Photo class is responsible for storing the information for each photo. This class is also responsible for storing the tags, dates, and captions for each photo. This class also has methods for getting the date of a photo. Author: Nile Kolenovic and Xander Kasternakis

Fields

path

The file path location of the photo on the user's machine.

caption

The caption associated with the photo.

tags

An ArrayList of strings representing tags associated with the photo.

date

A Calendar object representing the date of the last modification of the photo on the user's machine.

Constructor

Photo(String path)

The constructor takes in a path and creates a new photo object. The path is the file path location of the photo on the user's machine.

Parameters: String path - path of the photo

Author: Nile Kolenovic and Xander Kasternakis

Methods

getDate()

Gets the date of the photo object. The date refers to the last modification of the photo on the user's machine.

Returns: String - date of the photo object in the format MM/DD/YYYY HH:MM AM/PM

Author: Xander Kasternakis
