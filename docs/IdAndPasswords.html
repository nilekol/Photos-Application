IdAndPasswords Documentation

The IdAndPasswords class is responsible for storing the login information for each user. This class is also responsible for creating new accounts and checking if a login is valid. This class also stores the photo albums for each user. This class also has methods for reading and writing the login information and photo albums to a file stored in the data folder. This ensures data persistence between sessions. This class is also responsible for creating the admin and stock accounts. Author: Nile Kolenovic

Fields

loginInfo

A hashmap storing usernames and their respective passwords.

photoAlbums

A hashmap storing usernames and their respective photo albums.

stockCreated

A boolean indicating whether the stock account has been created.

Constructor

IdAndPasswords()

The constructor responsible for creating the admin and stock accounts. It checks if the loginInfo.ser file exists and creates it if not. Author: Nile Kolenovic

Methods

createAccount(String username, String password)

Creates new accounts by storing the provided username and password in the loginInfo hashmap. Also creates a new array list of photo albums for the new user. If the username or password is empty, the method returns. Author: Nile Kolenovic

Parameters: String username, String password - the username and password for the new account

checkLogin(String username, String password) : boolean

Checks if a login is valid by verifying the username and password against the loginInfo hashmap. Returns true if the login is valid, false otherwise. Author: Nile Kolenovic

Parameters: String username, String password - the username and password to check

Returns: boolean - true if the login is valid, false otherwise

getPhotoAlbums(String username) : ArrayList<PhotoAlbum>

Gets a user's photo albums. Author: Nile Kolenovic

Parameters: String username - the username of the user

Returns: ArrayList<PhotoAlbum> - the user's photo albums

updatePhotoAlbums(String username, ArrayList<PhotoAlbum> newAlbum)

Updates a user's photo albums. This can include adding/deleting photos or albums, renaming albums, and adding/removing tags/captions from photos. Author: Nile Kolenovic

Parameters: String username, ArrayList<PhotoAlbum> newAlbum - the username and the updated photo albums

writeToFile(IdAndPasswords idAndPasswords)

Writes the loginInfo hashmap to the loginInfo.ser file for data persistence. Prints "error" if there is an error in writing to the file. Author: Nile Kolenovic

Parameters: IdAndPasswords idAndPasswords - the IdAndPasswords object to write to the file

readToFile() : IdAndPasswords

Retrieves previous logins by reading the loginInfo.ser file and returning the IdAndPasswords object. This object contains login information and photo albums of all previous users for data persistence. Author: Nile Kolenovic

Returns: IdAndPasswords - the retrieved IdAndPasswords object
